# Digital Library Management System

## Overview

The Digital Library Management System is a Java-based application designed to manage books and library operations efficiently. The system allows users to add, search, issue, return, and display books through a simple console interface.

## Features

* Add New Books
* Search Books by Title or ID
* Display Available Books
* Issue Books
* Return Books
* Library Record Management
* User-Friendly Console Interface

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Collections Framework
* IntelliJ IDEA / Eclipse

## How to Run

1. Install Java JDK 21 or later.
2. Download or clone the project repository.
3. Open the project in IntelliJ IDEA or Eclipse.
4. Compile and run the main Java file.
5. Use the menu options to manage library operations.

* OOP Concepts
